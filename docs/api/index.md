# API Reference

Python module reference for zigporter. Useful if you want to use zigporter's
clients directly in your own scripts or extend its functionality.

| Module | Description |
|---|---|
| [`zigporter.models`](models.md) | Pydantic data models shared across the codebase |
| [`zigporter.ha_client`](ha_client.md) | Home Assistant WebSocket + REST client |
| [`zigporter.z2m_client`](z2m_client.md) | Zigbee2MQTT HTTP ingress client |
| [`zigporter.config`](config.md) | Configuration loading and environment helpers |
